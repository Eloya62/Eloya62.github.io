Installation machine virtuelle

Ce projet était une simulation de demande d'un client à une entreprise. Nous devions créer une machine virtuelle servant de support au développement d'un jeu vidéo et de rédiger tous les documents de support et de documentation liés à cette machine virtuelle. J'avais des consignes comme le système d'exploitation, la langue, les langages à installer…

Cette machine virtuelle est créée sur VMWare. J'ai dû chercher comment installer l'OS puis installer les langages et les lier entre eux.
